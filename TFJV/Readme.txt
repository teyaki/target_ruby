●はじめに

　TARGET frontier JVは、フリーウェアですので、送金登録等を行なう必要無く、
　インストール後は、そのまま使い続けることが可能です。

　DVD-ROM等からインストールした場合、すでに古いバージョンになってしまっている
　可能性が高いです。競馬のルールは毎年のように変更されていますし、また、
　Data Labでも、新しいデータが追加されたりしますので、それらへの対応や、
　新しい機能の追加等で、ターゲットは頻繁にバージョンアップしています。
　ですので、できるだけ早いタイミングでバージョンアップして下さい。
　最新のバージョンについては、下記のサポートサイトで情報が提供されています。
　また、メニューパネルの更新情報ボタンを押すと、最新バージョン等を
　自動的に確認できます。


●サポートについて

　TARGET frontier JVのサポートは、全てメール及びサポートサイトだけで
　行ないます。ですので、それらの手段が利用できない場合には、サポートを
　受けることが出来ないことを前提でのみご利用下さい。

　インターネットのTARGET公式サポートサイト

　　http://homepage2.nifty.com/~blitz/

　で情報を提供しています。

　また、JRA-VAN Data Lab,JV-Linkに関しては、

　　http://jra-van.jp/sup/index.html

　こちらのページを御覧下さい。


●基準タイムファイル

　TARGET frontier JVは様々な用途に使用することができますが、出馬表を
　使って予想する場合に、補正タイムを表示しますが、この補正タイムやTGXの
　各指数を表示するためには、最新の基準タイムファイルが必要になります。
　基準タイムファイルは、サポートホームページに登録されており、
　無料でダウンロードすることができます。画面左上のメインメニュー内の
　「基準タイムファイルのダウンロードと登録」を実行すれば、
　自動的に登録されます。ファイルの更新は基本的には毎週の月曜日の夜までに
　行なわれます。


●必要(推奨)動作環境

　TARGET frontier JVは、Windows XP/2000(32bit)/VISTA/7(32/64bit)上で
  動作いたします。また、Windows8上では、デスクトップ内でのみ動作いたします。

　CPUの能力は、Pentium 500MHz相当以上を推奨いたします。
　搭載メモリは、OSの推奨程度必要です。それ以下の場合は、正常に動作しない
　ことがあります。

　プログラム自体が占有するハードディスク容量は少ない(10Mバイト程度)ですが、
　JRA-VANのデータがかなりの容量を食いますので、できるだけ空き容量の大きい
　ドライブへインストールして下さい。できれば15Gバイトくらいの空き容量が
　欲しいところです。なお、プログラム用のフォルダとデータ用のフォルダは
　別々に指定できますので、とりあえずは10G程度の空きがあれば問題ありません。

　画面は1024x768以上が推奨ですが、それ以下でも一応動作はいたします。
　ただし、設定によっては、一部正常に表示されない箇所等が存在する可能性が
　ありますので、実際に試用して御確認下さい。

　また、インターネットにアクセスする機能を持っていますので、
　インターネットエクスプローラのVer4.0以上がインストールされている必要が
　あります。

　TARGET frontier JVは、その全てのデータをJRA-VANのData Labサービスを
　利用して取得し登録します。ですので、できればソフトを利用する前に、
　このサービスに加入して下さい。
　JRA-VANについては、
　http://www.jra-van.ne.jp/
　こちらを御覧下さい。


●アンインストール方法

　TARGET frontier JVのアンインストールはプログラムの削除とデータの削除の2段階
　あります。

　まず、コントロールパネルから、アプリケーションの追加と削除をダブルクリック
　して、TARGET frontier JVをクリックして追加と削除ボタンを押し、後は指示に
　従って下さい。これでプログラムが削除されます。

　あとは、インストールしたフォルダの中にデータが残っていますので、必要なものは
　保存し、最後にフォルダごと削除して下さい。以上でアンインストールは完了です。
